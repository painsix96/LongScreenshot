# App Store 截图规范与设计方案

## 截图规格要求

### iPhone 截图尺寸

| 设备类型 | 尺寸（像素） | 用途 |
|---------|-------------|------|
| **6.7英寸** | 1290 × 2796 | iPhone 15 Pro Max, 14 Pro Max |
| **6.5英寸** | 1242 × 2688 | iPhone 14 Plus, 13 Pro Max, 12 Pro Max |
| **5.5英寸** | 1242 × 2208 | iPhone 8 Plus, 7 Plus, 6s Plus |

### iPad 截图尺寸（可选但推荐）

| 设备类型 | 尺寸（像素） | 用途 |
|---------|-------------|------|
| **12.9英寸** | 2048 × 2732 | iPad Pro 12.9英寸 |
| **11英寸** | 1668 × 2388 | iPad Pro 11英寸, Air |

### 技术要求

- **格式**：PNG 或 JPEG（推荐 PNG 保留质量）
- **色彩空间**：sRGB 或 P3
- **文件大小**：单张不超过 50MB
- **数量**：每个设备组最多 10 张，建议 5 张
- **内容**：必须展示真实应用界面，禁止添加设备外壳

## 截图设计指南

### 布局原则

1. **F型视觉动线**：重要信息放在左上
2. **留白充足**：不要让界面元素过于拥挤
3. **层次分明**：通过大小、颜色区分信息层级
4. **一致性**：保持 5 张截图风格统一

### 配色建议

| 元素 | 建议颜色 | 用途 |
|-----|---------|------|
| 主色调 | #007AFF（iOS蓝）| 按钮、强调 |
| 辅助色 | #34C759（iOS绿）| 成功状态 |
| 背景 | #F2F2F7（系统灰）| 截图背景 |
| 文字 | #000000 / #FFFFFF | 根据背景调整 |

### 文案规范

- **字体**：苹方（PingFang SC）或 San Francisco
- **字号**：主标题 48-64pt，副标题 32-40pt
- **字数限制**：每行不超过 8 个汉字
- **语言**：简体中文（主），英文（国际版）

### 截图制作工具推荐

| 工具 | 用途 | 平台 |
|-----|------|------|
| **Figma** | UI 设计和截图排版 | Web/Mac |
| **Sketch** | UI 设计和截图排版 | Mac |
| **Xcode Simulator** | 获取真实截图 | Mac |
| **Screenshot Maker** | 批量生成截图 | iOS App |
| **AppLaunchpad** | 在线截图模板 | Web |

## 截图获取方法

### 方法一：Xcode Simulator

```bash
# 启动 Simulator
open -a Simulator

# 使用 xcrun 截图
xcrun simctl io booted screenshot screenshot.png
```

### 方法二：设备截图（推荐）

1. 在真机上运行应用
2. 使用快捷指令：
   - 带面容 ID：音量上 + 侧边按钮
   - 带主屏幕按钮：主屏幕按钮 + 侧边/顶部按钮
3. 通过 AirDrop 传输到 Mac

### 方法三：ScreenRecord 后裁剪

```bash
# 录制视频
xcrun simctl io booted recordVideo capture.mp4

# 使用 ffmpeg 截取帧
ffmpeg -i capture.mp4 -ss 00:00:05 -vframes 1 screenshot.png
```

## 截图检查清单

提交前请确认：

- [ ] 所有截图都是真实应用界面（非设计稿）
- [ ] 截图中没有测试数据或占位符
- [ ] 截图中不包含设备外壳
- [ ] 截图比例符合要求
- [ ] 截图清晰，无模糊或压缩失真
- [ ] 截图内容适合所有年龄段
- [ ] 截图中不显示状态栏的时间/运营商等敏感信息（可选）
- [ ] 截图中的文字没有拼写错误
- [ ] 5张截图有明确的叙事逻辑
- [ ] 截图在深色/浅色模式下都表现良好（如支持）

## 预览视频（可选）

如需制作 App Preview 视频：

| 项目 | 规格 |
|-----|------|
| **时长** | 15-30 秒 |
| **分辨率** | 与截图一致 |
| **格式** | H.264 编码的 MOV、M4V 或 MP4 |
| **帧率** | 30fps 或 60fps |
| **大小** | 不超过 500MB |

**视频内容建议**：
1. 0-5秒：展示首页和选择图片
2. 5-15秒：展示智能拼接过程
3. 15-25秒：展示标注编辑
4. 25-30秒：展示导出和分享
