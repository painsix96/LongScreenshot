## 应用描述（中文）

### 简短描述（促销文本）- 170字符以内
智能拼接聊天记录、网页截图，自动识别重叠区域，一键生成完美长截图。支持手动调整、标注编辑，保存高清原图。

### 完整描述

**长图拼接 - 智能截图拼接专家**

还在为拼接聊天记录、网页长截图而烦恼吗？长图拼接帮你轻松搞定！

【核心功能】
• **智能拼接**：采用先进图像识别算法，自动检测截图重叠区域，精准拼接无缝隙
• **批量处理**：支持一次选择多张截图，自动按顺序拼接成长图
• **手动调整**：提供可视化调整界面，支持拖拽对齐、精确微调
• **标注编辑**：内置画笔、箭头、矩形、马赛克、文字等多种标注工具
• **高清导出**：支持原图质量导出，不压缩画质，保留每一个细节

【使用场景】
✓ 微信、QQ、钉钉等聊天记录拼接
✓ 网页长截图保存
✓ 文档、邮件内容整理
✓ 购物清单、攻略分享
✓ 任何需要合并多张截图的场景

【特色功能】
• **智能识别**：自动分析图片内容，智能匹配最佳拼接位置
• **拼接预览**：实时预览拼接效果，所见即所得
• **历史记录**：自动保存拼接历史，随时查看和重新编辑
• **多种比例**：支持标准、方形、全屏等多种导出比例
• **隐私保护**：本地处理，图片不上传服务器，保护隐私安全

【操作简便】
1. 从相册选择需要拼接的截图
2. 应用自动识别并拼接
3. 根据需要调整或添加标注
4. 保存到相册或分享给好友

无需注册，无需登录，打开即用！

【用户反馈】
如有任何问题或建议，欢迎通过应用内"设置-反馈"联系我们，我们将持续改进产品体验。
